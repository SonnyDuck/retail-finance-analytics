# Business Requirement

## Project Overview

Marvolo Retail is a nationwide retail company operating over 100 stores across multiple cities. The company records hundreds of thousands of customer transactions every year, including sales, payments, shipments, promotions, and product returns.

Although the transactional database stores a large amount of operational data, management lacks a centralized analytical solution for monitoring business performance and supporting decision-making.

This project aims to transform raw transactional data into meaningful business insights using PostgreSQL and Power BI.

---

## Business Objectives

The management team would like to answer questions such as:

- How is overall sales performance changing over time?
- Which stores generate the highest revenue?
- Which product categories contribute the most to total sales?
- How effective are promotional campaigns?
- Which suppliers and products have the highest return rates?
- Which customer segments generate the most revenue?

---

## Project Scope

This project covers:

- Data exploration
- Data quality assessment
- Relational data modeling
- Business analysis using SQL
- Reporting layer
- Interactive Power BI dashboard
